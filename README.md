# Matlab Notification

Create windows notification for Matlab

## Use

There is a `notify-send.exe` file. Put this in your project directory ( or any folder you want for utilities ), then you can use `notify.m` function like this :

```matlab
notify('important', 'title', 'message');
notify('info', 'title', 'message');
notify('error', 'title', 'message');
```
